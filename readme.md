## Command to run
1. `npm init -y`
2. `npm install -D tailwindcss` -> create `index.css` inside src folder -> `@tailwind base;@tailwind components;@tailwind utilities;`
3. `npx tailwindcss@ init` -> config file: `tailwind.config.js`
4. `npx tailwindcss -i ./src/index.css -o ./dist/output.css --watch`
